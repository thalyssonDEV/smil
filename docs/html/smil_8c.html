int main ( )

Função principal do sistema.

A função main inicializa o hardware, configura os pinos de entrada e saída, e gerencia a lógica de funcionamento do sistema. Ela controla o estado do sensor, a ativação do modo noturno, o controle de brilho via joystick e a exibição dos dados no display, além de acionar o alarme sonoro quando necessário. A função executa continuamente enquanto o sistema está em operação.
